# Password Generator

A simple, interactive password generator web app built with **HTML**, **CSS**, and **JavaScript**.  
Users can customize password length and include uppercase letters, lowercase letters, numbers, and symbols. Includes a copy-to-clipboard feature.

- Include/exclude uppercase letters, lowercase letters, numbers, and symbols
- Randomly generates secure passwords
- Copy generated password to clipboard
- Validation messages for input and selections

## How to Use

1. Open the app in a web browser.
2. Set desired password length.
3. Select which character types to include.
4. Click **Generate** to create a password.
5. Click **Copy** to copy it to your clipboard.
